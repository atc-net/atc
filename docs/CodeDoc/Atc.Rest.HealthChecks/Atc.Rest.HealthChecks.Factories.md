#### CreateJson
>```csharp
>HealthCheckOptions CreateJson(string applicationName, JsonSerializerOptions jsonSerializerOptions, bool includeExceptionDetails = False)
>```
><b>Summary:</b> Creates health check options configured to return JSON formatted responses.
>
><b>Parameters:</b><br>
>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`applicationName`&nbsp;&nbsp;-&nbsp;&nbsp;The name of the application to include in the health check response.<br />
>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`jsonSerializerOptions`&nbsp;&nbsp;-&nbsp;&nbsp;Optional custom JSON serializer options. If null, default options are used.<br />
>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`includeExceptionDetails`&nbsp;&nbsp;-&nbsp;&nbsp;
            When , the JSON response includes raw exception messages from failing
            health checks. Defaults to  because exception messages can contain
            sensitive information (connection strings, file paths, internal hostnames). Only enable
            when the endpoint is restricted to trusted callers.
            <br />
>
><b>Returns:</b> A `Microsoft.AspNetCore.Diagnostics.HealthChecks.HealthCheckOptions` instance configured for JSON responses.
