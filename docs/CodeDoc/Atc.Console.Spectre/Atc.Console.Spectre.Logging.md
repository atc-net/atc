## ConsoleRenderingMode
Defines the rendering modes for console log output.

>```csharp
>public enum ConsoleRenderingMode
>```


| Value | Name | Description | Summary | 
| --- | --- | --- | --- | 
| 0 | Default | Default | Default rendering mode with minimal formatting. | 
| 1 | LogLevel | Log Level | Includes log level in the output. | 
| 2 | CategoryName | Category Name | Includes category name in the output. | 
| 3 | LogLevelAndCategoryName | Log Level And Category Name | Includes both log level and category name in the output. |
