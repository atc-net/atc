### Static Methods

#### AutoRegistrateServices
>```csharp
>void AutoRegistrateServices(this IServiceCollection services, Assembly interfaceAssembly, Assembly implementationAssembly)
>```
><b>Summary:</b> Automatically registers services by matching interfaces from one assembly with implementations from another.
>
><b>Parameters:</b><br>
>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`services`&nbsp;&nbsp;-&nbsp;&nbsp;The service collection.<br />
>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`interfaceAssembly`&nbsp;&nbsp;-&nbsp;&nbsp;The assembly containing service interfaces.<br />
>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`implementationAssembly`&nbsp;&nbsp;-&nbsp;&nbsp;The assembly containing service implementations.<br />
>
><b>Remarks:</b> This method scans both assemblies and registers each interface with its matching implementation as a transient service. Only the first matching implementation for each interface is registered.
